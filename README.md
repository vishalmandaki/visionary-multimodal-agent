# Visionary Multimodal Agent 🧠👁️

Next-generation AI agent capable of complex visual reasoning and zero-shot object perception using the latest Vision-Language Models.

## 🚀 Key Features
- **Visual Reasoning (VQA)**: Deep analysis of image content using LLaVA-style architectures.
- **Zero-Shot Perception**: Object tracking and identification without specific training.
- **Multimodal Integration**: Unified text and image processing for intelligent responses.

## 🛠️ Installation
```bash
git clone https://github.com/vishalmandaki/visionary-multimodal-agent.git
cd visionary-multimodal-agent
pip install -r requirements.txt
```

## 🚀 Quick Start
```python
from src.visual_reasoning_v2 import VisionaryReasoningAgent
agent = VisionaryReasoningAgent()
print(agent.ask_about_image("scene.jpg", "What is the primary action in this image?"))
```
